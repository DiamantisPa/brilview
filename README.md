# BRIL data inspector with web GUI

## Setup

1. build source distribution
   ```
   python setup.py sdist
   ```
   or 
   ```
   python setup.py sdist upload
   ```
2. installation

   locally
   
    ```
    python setup.py install
    ```
   
   or from pip repository
   
    ```
    pip install brilview
    ```
